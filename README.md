# Proyecto Reservas Pádel
Proyecto DAM – Sistema de gestión de reservas de pistas de pádel.

En este repositorio se incluyen los avances realizados en el proyecto intermodular de DAM correspondientes a la **segunda entrega (85%)**.

Se entrega un pdf con el contenido realizado hasta la fecha de la memoria: **Copia de Esqueleto_Memoria_TFG 85%**

El contenido del repositorio se organiza en las siguientes carpetas:

- **MySQL**: contiene los scripts SQL utilizados para la creación de la base de datos, las tablas y la inserción de datos de prueba.
- **Java**: incluye el código fuente del proyecto, con las clases modelo, las clases DAO, la lógica del sistema, la interfaz gráfica realizada con Java Swing y la clase `Main` utilizada para realizar pruebas de funcionamiento.
- **lib**: contiene el archivo `mysql-connector` que permite establecer la conexión entre la aplicación Java y la base de datos MySQL mediante JDBC.
- **Pruebas**: incluye una captura de pantalla de la consola donde se puede comprobar el correcto funcionamiento del programa tras ejecutar la clase `Main`.

El proyecto se encuentra en un estado avanzado de desarrollo (**85% completado**), incluyendo tanto la lógica de negocio como la interfaz gráfica funcional.
En la entrega final se corregirán los defectos existentes que puedan tener tanto la memoria como el código.
